# 🧠 Project Overview

SmartResume AI is a **full-stack final year project** that helps users:

✔ Build professional resumes  
✔ Analyze **ATS score**  
✔ Generate **career roadmaps**  
✔ Get **job suggestions**

Built with **Flutter + Django REST Framework + AI Logic**

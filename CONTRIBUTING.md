Contributing to PrepMate AI

Thank you for your interest in contributing to Prepmate AI 🚀

We welcome contributions from developers of all levels.

---

📌 How to Contribute

You can contribute in many ways:

- Fix bugs
- Improve UI
- Add new features
- Improve documentation
- Optimize backend APIs

---

🛠 Project Setup

Clone the repository

git clone https://github.com/provikash/prepMateAi.git

Backend Setup

cd backend
pip install -r requirements.txt
python manage.py runserver

Frontend Setup

cd frontend
flutter pub get
flutter run

---

🌿 Branch Strategy

We use the following branches:

Branch| Purpose
main| Production ready code
dev| Development branch
feature/*| New features
fix/*| Bug fixes

Always create a new branch from "dev".

Example:

git checkout dev
git checkout -b feature/resume-parser

---

📥 Pull Request Process

1. Fork the repository
2. Create a feature branch
3. Commit your changes
4. Push your branch
5. Create a Pull Request to "dev"

---

🧾 Commit Message Format

Use clear commit messages.

Examples:

feat: add ATS score analyzer
fix: resolve login API bug
ui: improve dashboard layout
docs: update README

---

🐛 Reporting Bugs

If you find a bug:

1. Open an issue
2. Describe the problem
3. Add screenshots if possible
4. Mention steps to reproduce

---

💡 Feature Requests

If you want to suggest a feature:

- Open a GitHub issue
- Explain the feature
- Explain the use case
